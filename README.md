
Analyzing NYC public school SAT performance using Python and pandas to identify top schools and borough-level trends.
This project uses Python and pandas to explore SAT performance across New York City public schools, using a dataset provided by DataCamp.

Analysis Questions

Which NYC schools have the best math results (scoring at least 80% of the maximum possible score)?
What are the top 10 schools based on combined SAT scores?
Which borough has the largest variation in combined SAT scores?

Key Features

Filtering and ranking schools based on performance thresholds
Creating new columns by combining existing data
Sorting and selecting top performers
Grouping data by borough and calculating descriptive statistics (mean, standard deviation)
Rounding and formatting numeric results

Technologies Used: Python, pandas
